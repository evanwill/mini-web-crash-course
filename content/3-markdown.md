# Markdown *(in a few minutes)*

[Markdown](https://daringfireball.net/projects/markdown/) is a standard to simplify writing content for the web. 
[GitHub markdown flavor](https://help.github.com/articles/basic-writing-and-formatting-syntax/) can be used any where on GitHub, in popular static site generators such as Jekyll, and to create documents using Pandoc.

- [Markdown in a Minute](https://evanwill.github.io/_drafts/notes/markdown-minute.html)
- GitHub Guide [Mastering Markdown](https://guides.github.com/features/mastering-markdown/)
